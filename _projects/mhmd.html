--- title: "Horizon 2020 Project: «My Health - My Data»" link: "http://www.myhealthmydata.eu/" year: 2017 category: research ---

My Health My Data (MHMD) is a Horizon 2020 Research and Innovation Action which aims to change the way sensitive data are shared.

Our goal at ATHENA RC was to build a Secure Multiparty Computation (SMPC) Infrastructure where a researcher can request for secure computations over specific datasets. The system is composed of three main entities: coordinator, smpc node, and data provider node. The components are sychronized by the coordinator by running a distributed protocol.

Our core MPC system lies in SCALE-MAMBA. SCALE-MAMBA assumes that the data are in the same server with the secure computation node meaning that external data providers are not supported. Having the ability to support external data providers is crucial as light nodes can participate to a computation request without the need to have heavy infrastructure to deploy and maintain a computation node. Based on Confidential Benchmarking based on Multiparty Computation we extend SCALE-MAMBA to support external data providers. Data providers send their data encrypted and each computation node takes a share (based on a secret share scheme) of the data. Νο computation node have access to the raw data. Our protocol is designed and developed as a finite state machine where client and server change states and act based on various events. The protocol is implemented in C++.

The work in this project was supported by "Athena" RC and partially supported by the European Union’s Horizon 2020 research and innovation programme under grant agreement No. 732907 (My Health, My Data).

Build with: Node.js, React, Python, WebSockets, C++ and SCALE-MAMBA (SMPC).
